# 更新日志

本项目遵循语义化版本（SemVer）。

## v0.3.0 - 2026-06-19

路由日志全面升级为可观测性 v2；下线 WebSocket 服务；新增客户端 CLI 版本检测与锚定升级；并修复一批 P0 安全与数据正确性问题。

### ✨ 新功能

#### 路由日志 · 可观测性 v2
- **统一数据模型**：明细落库 / 查询投影 / 实时推送共用一套 `RoutingEvent`，新增按「平台·组合 / 模型 / 账号 / 状态」四维度的日桶聚合，统一此前三个割裂的数据出口。
- **多维分析**：KPI（总请求 / 峰值 RPM / 成功率 / 平均延迟与 P95 / Token / 降级 / 命中组合）+ 请求趋势图 + 维度下钻 + Top 错误。
- **检索与详情**：keyset 分页 + 全维度过滤 + 关键字搜索（取代旧的固定 200 条无分页），新增请求详情抽屉与 CSV 导出。
- **实时模式**：实时尾巴注入明细的同时按节流重拉统计，修复「实时模式下统计与明细不同步」。

#### 客户端接入
- **CLI 版本检测与锚定升级**：检测各 IDE / CLI 客户端真实版本；升级命令按实际命中的安装位置（npm / bun / volta / brew 等）生成绝对路径，避免「升级了但版本没变」；升级后二次校验版本是否真正前进；多处安装时给出诊断与确认提示。

### 🐛 修复
- **P0 安全 / 正确性**：
  - skill ZIP 安装增加三重路径穿越防护 + 条目 / 字节上限，并修复「只返回不落库、重启即丢失」。
  - 凭证注入解析失败不再静默覆盖用户配置；`hosts.json` 改为 read-merge-write 保留其他 host。
  - 活动增量扫描仅在成功读取后推进 watermark，杜绝读失败导致的永久漏计。
  - kiro 计费优先透传上游真实 token（含 cache 读 / 写），仅缺失时才降级估算，修正用量 / 计费失真。
  - 自动更新 IPC 订阅改引用计数共享，消除重复监听。
- **relay 上游错误透传**：第三方 relay 上游错误此前被笼统落为 500 且丢失真实状态 / 消息；改为 4xx 原状态透传、5xx→502，并保留上游原始错误消息。

### 🔧 优化
- **下线 WebSocket 服务**：移除 push-server 上下文及其 IPC / 设置项（ws_port）/ 高级设置 UI 与文案。
- **统一反代缓存用量口径**：`Usage.inputTokens` 统一为「非缓存新增输入」，缓存读 / 写单列并在出站补回总输入。
- **Kiro 身份核对改按平台开关**：全局 `allowStaleKiroImport` 改为 per-platform `requireOnlineIdentityCheck`（默认关＝以占位身份导入、不联网；开启＝必须联网核对，失败即中止）。

### 📌 说明
- 自动更新经 GitHub Releases 审核后推送；macOS 当前未签名，采用「下载 dmg → 拖入应用程序」手动安装方式。

## v0.2.0 - 2026-06-17

自首个正式版 v0.1.0 以来的首个功能版本。核心是把**本地反代（账号池）**做成完整可用的能力，并对账号管理与整体交互做了体验升级。

### ✨ 新功能

#### 本地反代 · 账号池
- **账号池管理**（路由服务 → 账号池监控）：账号一键加入/移出池；每账号**优先级**与**并发上限**可调；支持批量设置 **429 限流冷却**。
- **选号策略**：sticky-lru（会话亲和）/ round-robin 可选，亲和 TTL 可配。
- **账号池韧性 · 按错误类型差异化处理**（避免反复撞墙、空烧额度）：
  - **402 配额耗尽** → 解析该账号「下一次配额重置时间」，冷却到那一刻再放行（缓存优先 + 在线兜底）。
  - **401/403 token 永久失效**（无 refreshToken / invalid_grant / api_key 模式）→ 自动移出反代池；**临时失败**（刷新 429 / 网络抖动）只短冷却稍后重试，不误删。
  - **429 限流** → 每账号可配置短冷却，与配额长冷却区分。
- **路由日志分析** + **Prometheus `/metrics`**：账号运行态（available / cooldown / rate_limited / quota_exhausted / suspended）一目了然。
- **kiro 实时模型目录**：按「会员档位最高」的可用账号拉取真实可用模型清单。

#### 账号管理
- 新增 **「会员计划」筛选**（按缓存的会员档位过滤；选项与卡片 / 表格展示同源）。
- 表格视图改为 **表格内部滚动 + 表头吸顶**（类 Ant Design）；账号 / 分组 / 代理 / 账号池表格统一。

#### 交互体验
- **路由 KeepAlive**：切换页面不再丢失筛选、搜索、选中等状态及滚动位置（类 Vue `<keep-alive>`）。
- **账号池监控页**：头部固定不滚动、内容区独立滚动，并新增按邮箱 / 账号 ID 搜索。

### 🔧 优化
- UI 组件对齐官方 shadcn：下拉（Select）恢复官方弹出形态；开关（Switch）改用官方 radix 实现。
- 仪表盘「会话数」补充说明：为本机磁盘**实时**会话文件数，会随各工具的会话保留期清理而增减（非历史累计）。
- 工程质量：启用 `exactOptionalPropertyTypes` / `noImplicitOverride`，统一 ESLint/Prettier，新增测试覆盖率门槛。

### 📌 说明
- 自动更新经 GitHub Releases 审核后推送；macOS 当前未签名，采用「下载 dmg → 拖入应用程序」手动安装方式。
